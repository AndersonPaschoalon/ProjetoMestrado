Set and stochastic model. Set a stochastic model, with its name (enum stochastic_functions type, std::string or C string format), first and second parameters, and AIC and BIC criterion. If there is no second parameter, it should be filled with zero. Currently it supports the follow distributions:

  • Weibull distribution

    \[ f(t) = \begin{cases} \frac{\alpha}{\beta^\alpha}t^{\alpha - 1}e^{(t/\beta)^{\alpha}}; & t \geq 0 \\ 0; & t < 0 \end{cases} \]

  • Normal distribution

    \[ f(t) = \frac{1}{\sqrt[]{2\sigma^2}\pi}e^{\frac{(t - \mu)^2}{2\sigma^2}} \]

  • Exponential distribution

    \[ f(t) = \begin{cases} \lambda e^{-\lambda t} ; & t \geq 0 \\ 0; & t < 0 \end{cases} \]

  • Pareto distibution

    \[ f(t) = \begin{cases} \frac{ \alpha t_{m}^\alpha}{t^{\alpha + 1}} ;& t \geq t_{m} \\ 0;& t < t_{m} \end{cases} \]

  • Cauchy distribution

    \[ f(t) = \frac{1}{\pi \gamma}[\frac{\gamma^2}{(t - t_{0})^{2} + \gamma^{2}}] \]

    On each case, the parameter order follow its alphabetical order. The current relationship between parameters are:
  • Weibull: \( (\alpha, \beta ) = (shape, scale) = (param1, param2)\);
  • Normal: \( (\mu, \sigma = (mean, standard deviation) = (param1, param2)\);
  • Exponential \( (\lambda ) = (rate) = (param1)\);
  • Pareto: \(( \alpha, x_m) = (shape, scale) = (param1, param2) \);
  • Cauchy: \((\gamma, x_0) = (scale, location) = (param1, param2)\).

Notice that on Cauchy distribution the first parameter is the scale!
